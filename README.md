# Linearization and Newton's Method
A reusable MATLAB workflow that computes the tangent-line approxiamtion for any function $f(x)$ at a point $x=a$. Demostration scripts and plots included.

---

**On Linearization**

For a differentiable function, $f(x)$, at a point $x=a$, the first-order Taylor approximation is: 

$$L(x) = f(a)+f'(a)(x-a)$$

Which coincides with the tangent line to the graph of $f$ at the point $(a,f(a))$
